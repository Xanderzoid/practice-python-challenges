## Milestone Year Calculator

A lightweight Python command-line utility that calculates exactly when a user will turn 100 years old.

### 1. Description
This tool accepts a user's name and age, then references the current system date to determine the target year. It is designed with modularity and the Unix design princples in mind.

### 2. Installation & Usage

1. Requirement: 
	- Python 3.6+

2. Run the script:
	- python script.py

3. Example Interaction:
	```text
   Enter your name: Xander
   Enter your age: 32
   Hello, Xander. You will turn 100 in the year 2094
